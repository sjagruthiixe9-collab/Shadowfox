# 📊 Data Visualization using Pandas and Seaborn

## 📌 Overview
This task focuses on exploring data visualization techniques using Pandas and Seaborn. It demonstrates how different types of plots can be used to understand data patterns, distributions, and relationships.

- Pandas  
- Seaborn  
- Matplotlib

- Histogram  
- Pie Chart  
- Box Plot  

### 🔹 Seaborn Plots
- Line Plot  
- Bar Plot  
- Scatter Plot  
- Histogram  
- Box Plot  
- Heatmap

## ⚙️ Implementation
- Used Pandas for basic plotting and data handling  
- Used Seaborn for advanced and visually appealing statistical plots  
- Applied different plot types to understand trends and distributions
